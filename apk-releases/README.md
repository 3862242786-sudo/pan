## 安装说明

1. 下载对应版本的 APK 文件
2. 在安卓手机上安装
3. 如提示"未知来源应用"，请在设置中允许

## 系统要求
- Android 7.0 (API 24) 及以上
- 建议 Android 10+ 以获得最佳体验
