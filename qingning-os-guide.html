01 青柠OS 是什么？

青柠OS（QingningOS）是一个基于 Debian Linux 的桌面操作系统，专为虚拟机环境设计。它采用了 Google 的 Material Design 设计语言，拥有现代化的深色主题界面。

与其他 Linux 发行版不同的是，青柠OS 提供了完整的自动化构建脚本，你只需要运行一条命令，就能生成属于自己的可启动 ISO 镜像。

💡 适合人群
本教程适合所有对 Linux 感兴趣的用户，无论你是初学者还是有经验的用户，都能轻松上手。不需要编程经验！

系统特点

  • 基于 Debian 稳定版 - 享受长期安全更新支持
  • Material Design 主题 - 现代化的深色界面，视觉体验出色
  • 轻量级 XFCE4 桌面 - 资源占用低，虚拟机中流畅运行
  • 预装常用软件 - Firefox、LibreOffice、VLC 等开箱即用
  • Powered by Linux - 开源免费，自由定制

02 准备工作

在开始之前，请确保你满足以下条件：

硬件要求

  • 一台能上网的电脑（Windows、macOS 或 Linux 均可）
  • 至少 10GB 的可用磁盘空间
  • 至少 4GB 内存（推荐 8GB）
  • 支持虚拟化的 CPU（Intel VT-x 或 AMD-V）

软件要求

  • 虚拟机软件：VMware Workstation Player（免费）或 VirtualBox（免费开源）
  • Linux 构建环境：一台安装了 Debian 12 或 Ubuntu 22.04+ 的电脑/虚拟机
  • root 权限：构建脚本需要管理员权限
⚠️ 重要提示
构建 ISO 需要在 Linux 环境下进行。如果你没有 Linux 电脑，可以先用 VirtualBox 安装一个 Ubuntu 虚拟机，然后在里面运行构建脚本。

检查虚拟化支持

在 Windows 上，按 Win + R，输入 msinfo32 回车。在系统信息中查找"Hyper-V - 虚拟机监视器模式扩展"，如果显示"是"，说明支持虚拟化。

03 下载源代码

青柠OS 的源代码以 ZIP 压缩包形式提供，包含了构建系统所需的所有文件。

1

下载源码包

点击下方的下载按钮，获取青柠OS v1.0.0 的完整源代码：

下载青柠OS v1.0.0 源码
2

解压文件

将下载的 ZIP 文件解压到你的 Linux 构建环境中。你可以使用以下命令：

$ unzip qingning-os-v1.0.0.zip -d ~/qingning-os Archive: qingning-os-v1.0.0.zip inflating: qingning-os/build.sh inflating: qingning-os/config/setup-chroot.sh ...
3

查看目录结构

解压后，你会看到以下目录结构：

$ cd ~/qingning-os && ls -la drwxr-xr-x build.sh # 主构建脚本 drwxr-xr-x config/ # 系统配置文件 drwxr-xr-x docs/ # 文档 drwxr-xr-x scripts/ # 辅助脚本 drwxr-xr-x theme/ # Material Design 主题 -rw-r--r-- README.md # 说明文档 -rw-r--r-- LICENSE # 开源协议

04 构建 ISO 镜像

这是最关键的一步。青柠OS 提供了一个全自动的构建脚本，你只需要运行它，等待几分钟，ISO 镜像就会自动生成。

1

安装构建依赖

在构建之前，需要安装一些必要的工具。在 Debian/Ubuntu 上运行：

$ sudo apt update $ sudo apt install -y debootstrap xorriso isolinux grub-pc-bin grub-efi-amd64-bin squashfs-tools dosfstools mtools

这些工具的作用：

  • debootstrap - 创建 Debian 基础系统
  • xorriso - 生成 ISO 镜像
  • grub-efi-amd64-bin - UEFI 启动支持
  • squashfs-tools - 压缩文件系统
2

运行构建脚本

给脚本添加执行权限，然后以 root 身份运行：

$ cd ~/qingning-os $ chmod +x build.sh $ sudo ./build.sh [QingningOS] 检查 root 权限... 通过 [QingningOS] 安装构建依赖... [QingningOS] 创建 4GB 磁盘镜像... [QingningOS] 引导 Debian Bookworm... [QingningOS] 配置系统... [QingningOS] 安装软件包（这可能需要几分钟）... [QingningOS] 应用 Material Design 主题... [QingningOS] 创建 ISO 镜像... [QingningOS] 完成！ISO: output/qingning-os-1.0.0-amd64.iso
3

验证构建结果

构建完成后，检查 ISO 文件是否生成：

$ ls -lh output/*.iso -rw-r--r-- 1 root root 1.2G Jun 6 10:00 output/qingning-os-1.0.0-amd64.iso

ISO 文件大小通常在 1GB 到 2GB 之间，具体取决于安装的软件包数量。

💡 构建时间
首次构建需要下载大量软件包，根据网络速度，可能需要 15-30 分钟。后续构建会快很多，因为软件包已经缓存。

05 在 VMware 中安装

VMware Workstation Player 是 VMware 提供的免费虚拟机软件，非常适合运行青柠OS。

安装 VMware Workstation Player

  1. 访问 VMware 官网 下载 Workstation Player
  2. 运行安装程序，按照向导完成安装
  3. 安装完成后重启电脑

创建虚拟机

1

打开 VMware，创建新虚拟机

点击"创建新虚拟机"，选择"典型（推荐）"配置。

2

选择 ISO 镜像

选择"安装程序光盘映像文件(iso)"，然后浏览到你构建的 ISO 文件：

~/qingning-os/output/qingning-os-1.0.0-amd64.iso
3

选择操作系统类型

客户机操作系统选择 Linux，版本选择 Debian 12.x 64位。

4

配置虚拟机硬件

建议配置：

  • 处理器：2 个核心
  • 内存：2048 MB（2GB）
  • 硬盘：20 GB，拆分为多个文件
  • 网络：NAT（默认即可）
5

启动虚拟机

点击"完成"创建虚拟机，然后点击"开启此虚拟机"。

你会看到青柠OS 的 GRUB 启动菜单，显示 "Powered by Linux" 标识。选择第一个选项启动系统。

06 在 VirtualBox 中安装

VirtualBox 是 Oracle 提供的免费开源虚拟机软件，同样完美支持青柠OS。

安装 VirtualBox

  1. 访问 VirtualBox 官网 下载
  2. 同时下载"VirtualBox Extension Pack"（增强功能包）
  3. 安装 VirtualBox 主程序，然后双击 Extension Pack 安装

创建虚拟机

1

新建虚拟机

点击"新建"按钮，输入名称"QingningOS"，类型选择 Linux，版本选择 Debian (64-bit)。

2

分配内存

建议分配 2048 MB（2GB）内存。如果你的电脑内存充足，可以分配 4096 MB。

3

创建虚拟硬盘

选择"现在创建虚拟硬盘"，文件类型选择 VDI，存储选择"动态分配"，大小设为 20 GB。

4

加载 ISO 镜像

创建完成后，选中虚拟机，点击"设置" → "存储" → 在"控制器: IDE"下点击光盘图标 → "选择虚拟盘" → 选择你的 ISO 文件。

5

启动系统

点击"启动"按钮。看到 GRUB 菜单后选择第一个选项进入系统。

首次启动会自动进入桌面，用户名为 qingning，密码为 qingning。

💡 安装增强功能（推荐）
在 VirtualBox 中运行青柠OS 后，建议安装"增强功能"以获得更好的体验（共享剪贴板、自动调整分辨率等）。在菜单栏选择"设备" → "安装增强功能"。

07 首次启动与配置

成功启动青柠OS 后，你会看到 Material Design 风格的登录界面和桌面。

登录系统

  • 用户名：qingning
  • 密码：qingning
  • 系统已配置自动登录，通常不需要手动输入

欢迎界面

首次启动时，系统会显示一个终端窗口，展示青柠OS 的 ASCII Logo 和系统信息：

基本设置

  1. 分辨率：如果屏幕分辨率不合适，右键桌面 → 应用程序 → 设置 → 显示，选择合适的分辨率
  2. 网络：默认已启用 NetworkManager，点击面板右侧的网络图标即可连接 WiFi
  3. 时区：系统默认使用上海时区（CST），如需更改，打开终端输入 sudo dpkg-reconfigure tzdata
  4. 语言：系统已预装中文支持，界面默认为中文

08 桌面环境使用指南

青柠OS 使用 XFCE4 桌面环境，经过 Material Design 主题定制，界面现代美观。

桌面布局

  • 顶部面板：左侧是应用程序菜单（Whisker Menu），中间显示当前打开的窗口，右侧是系统托盘（网络、音量、时间）
  • 桌面：深色背景，中央可放置常用快捷方式
  • 底部面板（可选）：任务栏，显示所有打开的窗口

常用操作

操作 方法
打开应用程序菜单 点击左上角"青柠"图标，或按 Super 键（Windows键）
打开终端 按 Ctrl + Alt + T
切换工作区 按 Ctrl + Alt + ←/→，或点击面板上的工作区切换器
截图 按 Print Screen 键
锁定屏幕 按 Ctrl + Alt + L
注销/关机 点击右上角电源图标

文件管理器（Thunar）

Thunar 是 XFCE4 的默认文件管理器，支持：

  • 标签页浏览（按 Ctrl + T 打开新标签）
  • 分栏视图
  • 压缩/解压（右键菜单）
  • 自定义操作

09 预装应用介绍

青柠OS 预装了一系列常用软件，让你开箱即用。

网络与办公

  • Firefox ESR - 网页浏览器，稳定可靠
  • Thunderbird - 电子邮件客户端
  • LibreOffice - 办公套件（Writer文字处理、Calc电子表格、Impress演示文稿）

多媒体

  • VLC - 万能媒体播放器，支持几乎所有视频格式
  • Ristretto - 图片查看器
  • Parole - XFCE 媒体播放器

系统工具

  • GNOME Disks - 磁盘管理工具
  • Baobab - 磁盘使用情况分析器
  • Neofetch - 系统信息展示工具
  • htop - 交互式进程查看器

安装更多软件

使用 apt 包管理器安装软件：

$ sudo apt update $ sudo apt install 软件包名称

例如，安装 GIMP 图像编辑器：

$ sudo apt install gimp

10 自定义主题

青柠OS 提供了完整的 Material Design 主题，你也可以根据个人喜好进行调整。

更换壁纸

  1. 右键桌面 → "桌面设置"
  2. 在"背景"选项卡中，点击"文件"选择你喜欢的图片
  3. 或者选择系统自带的青柠OS 默认壁纸

调整主题颜色

如果你想修改主题颜色，可以编辑 GTK3 主题文件：

$ nano ~/.config/gtk-3.0/gtk.css

修改 CSS 变量中的颜色值，例如将 #22c55e（绿色）改为 #3b82f6（蓝色）。

添加桌面小部件

XFCE 支持多种面板小部件：

  1. 右键面板 → "面板" → "面板首选项"
  2. 点击"项目"选项卡
  3. 点击"+"按钮添加小部件（如天气、系统监视器、时钟等）

11 常见问题排查

构建失败

问题：运行 build.sh 时提示权限不足
解决：确保使用 sudo 运行脚本，且当前用户在 sudoers 列表中。
问题：debootstrap 下载速度慢或失败
解决：修改 build.sh 中的 MIRROR 变量，使用国内镜像源：
MIRROR="https://mirrors.tuna.tsinghua.edu.cn/debian"

虚拟机相关问题

问题：虚拟机无法启动，提示"VT-x is disabled"
解决：进入 BIOS/UEFI 设置，启用 Intel VT-x 或 AMD-V 虚拟化技术。
问题：屏幕分辨率无法调整
解决：在 VirtualBox 中安装增强功能；在 VMware 中安装 VMware Tools。
问题：没有网络连接
解决：检查虚拟机网络适配器设置，确保选择了 NAT 模式。在终端运行 sudo systemctl restart NetworkManager。

系统使用问题

问题：忘记密码
解决：在登录界面按 Ctrl + Alt + F1 进入 TTY，以 root 身份登录（密码也是 qingning），然后运行 passwd qingning 重置密码。
问题：中文输入法无法使用
解决：青柠OS 默认安装了 fcitx5 输入法框架。按 Ctrl + Space 切换输入法。如需添加更多输入法，运行 sudo apt install fcitx5-chinese-addons。
