青柠游戏内核 (QGE) 是一个专为 HTML5 游戏优化的轻量引擎。v2.0 新增 Three.js 3D 渲染支持，2D 零依赖、3D 集成化，让你专注于游戏创意。

功能特性

从零开始构建，为游戏开发量身定制

60fps 游戏循环

基于 requestAnimationFrame 的高精度游戏循环，自动计算 deltaTime，内置 FPS 计数器，确保流畅的游戏体验。

精灵与动画

完整的 Sprite 系统，支持位置、旋转、缩放、锚点、透明度。内置帧动画和补间动画，丰富的 Easing 函数。

碰撞检测

支持 AABB、圆形、矩形-圆形多种碰撞检测方式。可选像素级精确检测，满足从休闲到动作游戏的各种需求。

粒子系统

高性能粒子发射器，支持重力、生命周期、颜色渐变、旋转。内置对象池，千级粒子不卡顿。

场景管理

场景栈管理，支持 push/pop/switch 操作。每个场景拥有完整的生命周期：init、update、render、pause、resume、destroy。

摄像机系统

跟随目标、边界限制、屏幕震动、缩放控制。支持世界坐标与屏幕坐标转换，轻松实现大地图滚动。

统一输入

键盘、鼠标、触摸事件统一处理。支持每帧瞬时状态检测（pressed/released），移动端虚拟摇杆内置。

3D 渲染

集成 Three.js，支持 3D 模型、光照、阴影、雾效、粒子。按需加载，2D 游戏不受影响。

零依赖 (2D)

纯原生 JavaScript，不依赖任何第三方库。单文件引入即可使用，gzip 后不到 30KB。

快速开始

5 行代码启动你的第一个游戏

// 1. 引入引擎
<script src="qge/qge-core.js"></script>

// 2. 创建游戏实例
const game = new QGE.Core({
    canvas: document.getElementById('game'),
    width: 800,
    height: 600
});

// 3. 创建场景
class PlayScene extends QGE.Scene {
    init() {
        this.player = new QGE.Sprite({ x: 400, y: 500, color: '#22c55e' });
        this.addChild(this.player);
    }
    update(dt) {
        if (QGE.input.isKeyDown('ArrowLeft')) this.player.x -= 200 * dt;
        if (QGE.input.isKeyDown('ArrowRight')) this.player.x += 200 * dt;
    }
}

// 4. 启动游戏
game.scenes.switch(new PlayScene());
game.start();
