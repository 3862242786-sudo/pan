☁️ 二、青柠网盘使用指南

青柠网盘是一个简洁高效的云端存储服务，无需注册即可使用。

2.1 访问网盘

步骤 1：打开官网

在浏览器中访问 https://3862242786-sudo.github.io/pan/

步骤 2：上传文件

2.2 Android APP

🛡️ 三、青柠杀毒（Windows）使用指南

3.1 安装

步骤 1：下载

从 青柠应用商店 或官网下载 青柠杀毒v6.0.exe

步骤 2：运行

双击运行EXE文件，程序会自动请求管理员权限（内置UAC提权）。

3.2 主要功能

  • 快速扫描：扫描系统关键目录和常见病毒位置
  • 全盘扫描：深度扫描所有磁盘（支持2MB大文件跳过）
  • 文件粉碎：安全删除敏感文件，防止恢复（64KB分块覆写）
  • 密码生成器：生成高强度随机密码
  • 系统检测：检查防火墙、更新状态、启动项等
提示：青柠杀毒使用本地病毒库，无需联网即可扫描。病毒库文件为 virus_db.json，可手动更新。

3.3 扩展模块

青柠杀毒支持扩展模块（.qnmod文件），可扩展功能：

  • 深色主题 / 浅色主题 / 赛博朋克主题
  • 启动项管理器
  • 进程管理器
  • 更多病毒库扩展

📱 四、青柠杀毒（Android）使用指南

4.1 安装

步骤 1：下载 APK

下载 青柠杀毒_v6.0.0.apk 到手机

步骤 2：允许安装

在设置中允许「安装未知来源应用」，然后安装APK

4.2 主要功能

  • APK扫描：扫描已安装和本地APK文件
  • 系统检测：Root检测、Bootloader解锁检测、权限检查
  • 病毒库：内置本地病毒特征库
  • 底部弹窗权限：优雅的权限申请方式，无需跳转设置
注意：Android版需要存储权限才能扫描APK文件。首次启动时会通过底部弹窗申请权限。

4.3 Bootloader检测

v6.0版本支持4层Bootloader解锁检测：

  1. 读取 ro.boot.verifiedbootstate
  2. 检查 ro.boot.flash.locked
  3. 检查 ro.secure 属性
  4. 分析 /proc/cmdline 参数

🖥️ 五、青柠桌面安装与使用

重要提示：青柠桌面会替换Windows默认桌面环境（Shell）。安装前请确保已创建系统还原点或VM快照。程序内置安全恢复机制，黑屏时会自动回退到Explorer。

5.1 安装步骤

步骤 1：下载

从应用商店下载 QingningDesktop.exe

步骤 2：右键管理员运行

右键EXE文件 → 以管理员身份运行。程序内置UAC提权，会自动请求最高权限。

步骤 3：点击安装

在弹出的安装对话框中点击「安装青柠桌面」，程序会：

  1. 备份当前Shell注册表值
  2. 将青柠桌面设置为默认Shell
  3. 创建恢复脚本

步骤 4：重启电脑

安装完成后必须重启电脑才能生效。

5.2 功能特性

  • Dock栏任务栏：类似苹果Dock的底部任务栏，居中显示，带悬停放大效果
  • 开机Logo：显示「青柠OS」Logo和「Powered by Windows」
  • 文件管理器：内置简洁文件管理，保留Windows资源管理器备用
  • 开始菜单：青柠风格应用启动器
  • SVG图标：所有图标均为矢量，清晰锐利

5.3 安全恢复

如果安装后遇到问题，有多种恢复方式：

自动恢复：程序内置异常处理，崩溃时自动写日志并恢复Explorer Shell。

手动恢复方法 1：运行恢复脚本

在桌面程序目录中找到 restore_desktop.bat，右键管理员运行。

手动恢复方法 2：注册表修复

按 Ctrl+Shift+Esc 打开任务管理器 → 运行新任务 → 输入 regedit → 定位到：
HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows NT\CurrentVersion\Winlogon
将 Shell 值改回 explorer.exe

手动恢复方法 3：安全模式

重启时按F8进入安全模式，系统会自动使用Explorer，然后卸载青柠桌面。

5.4 v2.0改进

  • 移除GDI+依赖，改用纯GDI绘制，兼容VMware
  • 添加SetUnhandledExceptionFilter异常捕获
  • 添加崩溃标志文件检测
  • 安全模式自动跳过
  • 分步启动，每步记录debug.log

❓ 八、常见问题解答（FAQ）

Q1：青柠桌面安装后黑屏怎么办？

A：v2.0已修复此问题。如果仍遇到黑屏，程序会在10秒后自动恢复Explorer。也可按 Ctrl+Alt+Del 打开任务管理器，运行 explorer.exe 临时恢复。

Q2：杀毒软件误报怎么办？

A：青柠杀毒是开源安全工具，由于使用了一些系统API（如注册表修改、文件遍历），可能被部分杀毒软件误报。请将程序添加到白名单。

Q3：Android版无法安装？

A：请检查：1）是否允许未知来源安装；2）APK是否下载完整；3）手机存储空间是否充足。

Q4：如何更新病毒库？

A：Windows版病毒库文件为 virus_db.json，替换即可。Android版病毒库在APK的assets目录中，需重新安装更新版APK。

Q5：青柠桌面支持Windows 11吗？

A：当前版本主要针对Windows 10优化。Windows 11可能兼容，但未经充分测试。

Q6：应用商店下载失败？

A：检查网络连接，或直接从官网下载对应EXE文件。下载链接使用Supabase存储，部分地区可能需要代理。
