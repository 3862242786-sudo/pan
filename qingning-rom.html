针对虚拟大师的兼容性优化

VM

虚拟大师 (Virtual Master)

一款 Android 虚拟机应用，可在手机内运行独立的 Android 系统。青柠 ROM 针对其虚拟化环境做了专门适配，确保系统服务、图形渲染、网络连接在虚拟机内正常运行。

Android 11 API 30 arm64-v8a 轻量精简

ROM 特性

为虚拟环境量身定制的系统优化

R

根权限内置

预装 Magisk 或 SuperSU，虚拟机内直接获取 root，无需额外操作。

X

Xposed 框架

预装 LSPosed 或 EdXposed，支持模块扩展，增强系统功能。

G

GApps 可选

提供带 Google 服务和不带 Google 服务两个版本，按需选择。

S

系统精简

移除厂商预装应用，减少后台服务，降低内存占用，提升虚拟机性能。

N

网络优化

修复虚拟机内 DNS 解析、WiFi 代理、VPN 兼容性等问题。

D

调试支持

开启 ADB 无线调试、日志记录，方便开发和排错。
