核心特性

按需加载
页面只加载需要的模块，不加载多余代码。核心 15KB，模块按需拉取。
智能缓存
LRU 缓存 + TTL 过期策略，站点设置等数据缓存 60 秒，减少重复请求。
统一认证
登录、注册、权限检查一套代码全站共享，导航栏自动更新，状态同步无延迟。
站点控制
站长一键关闭网站，自定义维护页面文字，激活码解锁，权限分层管理。
事件总线
组件间通信不用层层传参，直接 emit/on。登录登出事件自动触发页面刷新。
统一视觉
CSS 变量 + 组件类名，所有页面风格一致。暗色主题、玻璃效果、动画标准化。
统一状态层
Store 模块，跨标签页同步，登录状态实时共享。多标签页之间数据一致性保障。
性能监控
Monitor 模块，页面加载采集，API 计时，错误自动捕获。全链路性能可视化。
安全层
Security 模块，HMAC 签名验证，XSS 防护，输入消毒。为全站安全保驾护航。
插件系统
Plugin 模块，可扩展的插件注册机制，按需启用/禁用。灵活扩展，不影响核心。
