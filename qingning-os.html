安装指南

从源代码构建到虚拟机安装，只需四步

1

下载源代码

从 GitHub 克隆 QingningOS 仓库到本地

git clone https://github.com/3862242786-sudo/qingning-os.git
2

运行构建脚本

赋予构建脚本执行权限并以 root 身份运行

chmod +x build.sh && sudo ./build.sh
3

生成 ISO 镜像

构建完成后，ISO 文件将输出到 output 目录

output/qingning-os-1.0.0-amd64.iso
4

在虚拟机中安装

打开 VMware 或 VirtualBox，创建新虚拟机，选择生成的 ISO 镜像文件，分配 2GB+ 内存和 10GB+ 磁盘空间，启动并按照提示完成安装。
