📋 青柠网盘文件保护政策

了解我们如何保护您的文件安全，防止误删和数据丢失

🛡️ 什么是文件保护政策？

青柠网盘文件保护政策是一项自动化的文件安全措施。每当您上传文件时，系统会自动生成一个专属的确认文件（.qn-verify），作为该文件的"数字身份证"。

核心目标

防止文件被误删、误改，并在发生数据异常时提供恢复依据。

📄 确认文件包含什么？

每个确认文件都是一个 JSON 格式的文本文件，包含以下信息：

{
  "policy": "QINGNING_FILE_PROTECTION_POLICY",
  "version": "1.0",
  "originalFile": {
    "name": "您的原始文件名.pdf",
    "storedName": "1234567890_abc123.pdf",
    "size": 1048576,
    "uploadedAt": "2024-01-15T08:30:00.000Z"
  },
  "verification": {
    "status": "VERIFIED",
    "method": "QN_AUTO_VERIFY",
    "timestamp": 1705312200000,
    "message": "此文件已通过青柠网盘安全上传验证"
  },
  "protection": {
    "type": "FILE_INTEGRITY_PROTECTION",
    "description": "该确认文件用于防止原始文件被误删或篡改",
    "restoreInfo": "如原文件丢失，可凭此确认文件联系管理员恢复"
  }
}

⚙️ 工作原理

1. 文件上传

您选择文件并点击上传，文件被传输到青柠网盘服务器

2. 生成确认文件

系统自动创建包含文件信息的 .qn-verify 确认文件

3. 安全存储

原文件和确认文件同时存储，相互关联保护

4. 验证显示

文件列表中显示"✓ 已验证"徽章，表明受保护状态

🚨 文件恢复流程

如果您的文件不幸被误删或损坏，可以通过以下步骤申请恢复：

  • 查找确认文件：在文件列表中找到同名但后缀为 .qn-verify 的确认文件
  • 下载确认文件：保存该文件到本地，其中包含恢复所需的所有信息
  • 联系管理员：通过关于页面的联系方式，向管理员提供确认文件
  • 等待恢复：管理员核实信息后，将尽力恢复您的原始文件

重要提示

确认文件本身不包含原始文件内容，仅包含元数据信息。如果原文件和确认文件同时丢失，将无法恢复。

❓ 常见问题

确认文件会占用额外空间吗？

确认文件通常只有几百字节到几千字节，占用空间极小，对您的存储配额影响可以忽略不计。

我可以删除确认文件吗？

可以，但不建议。删除确认文件后，该文件将失去保护，被误删后无法恢复。

确认文件包含我的隐私信息吗？

不包含。确认文件只包含文件名、大小、上传时间等基础元数据，不包含文件内容本身。

所有文件都会生成确认文件吗？

是的，从政策实施之日起，所有通过青柠网盘上传的文件都会自动生成对应的确认文件。
