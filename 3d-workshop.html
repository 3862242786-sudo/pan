下载

获取源码，在本地构建运行

Windows EXE

Windows 10/11 本地应用，基于 Python + PyQt6 构建，提供完整的图形界面操作体验。

系统要求：Windows 10/11, Python 3.10+
下载源码

Android APK

Android 应用，生成 Termux 命令执行，在手机上即可完成 3D 转换。

系统要求：Android 8.0+, Termux
下载源码

使用指南

四步完成 2D 到 3D 的转换

1

选择模式

根据需求选择 SHARP 单图3D、DUSt3R 多视图或 Depth Pro 深度图模式

2

上传素材

选择一张或多张照片作为输入素材，支持常见图片格式

3

配置参数

选择运行设备（GPU/CPU）、输出分辨率等参数

4

生成3D

一键生成3D场景，支持在线预览和下载结果
